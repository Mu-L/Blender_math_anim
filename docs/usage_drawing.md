# Free Drawing
Organize your drawings in layers and use the full power of grease pencil to create freehand drawings and animations.

1. Add/Select drawing holding object and layers, then enter **`Draw Mode`**
2. Use mouse or tablet to draw or write stuffs.
   - Remember to uncheck `Use Custom Radius` both in `Object Settings` and `Layer Settings` if you want to draw 
     with pressure size.
   - Don't forget to exit `Draw Mode` when finished.
3. Supports:
   - All native grease pencil tools and settings
   - Can also manually adjust some controls 
   - Grease pencil effect modifiers
4. When add/delete drawing layers, you need to `Update Layer Tree` manually to ensure they are in sync.

## Animation
- Preset animations can easily apply to Drawing layers.
- Morph between and across drawings and other types
