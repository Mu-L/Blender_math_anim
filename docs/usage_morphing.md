# Morphing Setups

You can morph between:
- Functions ↔ Functions  
- Text ↔ Text  
- Drawing ↔ Drawing  
- Function ↔ Drawing (after sampling)

## Steps
1. Select **A** and **B**
2. Click: **Create Morph**
3. A morph object appears with:
   - resolution
   - easing type
   - duration
4. Animate morph value from 0 → 1.
