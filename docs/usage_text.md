# Formula Generating and Animating

You can create formula objects rendered using Typst or OpTeX.

1. Add → Math Anim → **Formula**
2. Choose engine:
   - Typst
   - OpTeX / LaTeX
3. Enter input text: $ \int_0^\pi \sin(x),dx = 2 $
4. Render

## Animating Formulas
- Writing animation
- Morphing into other formulas
- Moving along paths
